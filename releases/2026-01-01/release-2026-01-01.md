### 1.OpenWorkers: 自托管的 V8 JavaScript 运行时

![OpenWorkers: 自托管的 V8 JavaScript 运行时](https://showhntoday.com/images/46454693.png)

**URL：** [https://openworkers.com/introducing-openworkers](https://openworkers.com/introducing-openworkers)

**作者：** max_lt

**描述：**
OpenWorkers 是一个允许你在自己的服务器上运行不受信任的 JavaScript 代码的项目，其体验与 Cloudflare Workers 类似，但避免了供应商锁定。它使用了 V8 JavaScript 引擎的隔离（isolates）特性，并用 Rust 语言编写，以确保安全性和高性能。该项目支持 fetch（网络请求）、KV（键值存储）、Postgres 数据库连接、S3/R2 对象存储、定时任务（cron）以及加密 API。你只需要一个 Docker Compose 文件和 PostgreSQL 数据库即可轻松部署。

**如何使用：**
开发者可以将 OpenWorkers 部署在自己的服务器上，通过一个简单的 Docker Compose 文件即可完成。部署完成后，你可以编写 JavaScript 代码，然后将其上传到 OpenWorkers 运行。这非常适合那些希望拥有 Cloudflare Workers 的便利性，但又不希望被特定云服务商绑定的场景。例如，你可以用它来处理 Webhook 请求、构建微服务、运行定时任务清理数据，或者提供一个可定制的 API 接口。

**如何使用：**
