### 4.Sklad 仓库：离线优先的开发者数据仓库

**作者：** rench321

**描述：**
Sklad 是一个利用 Rust 和 Tauri v2 构建的应用程序，旨在为开发者提供一个安全、离线优先的片段管理器。它解决了开发者在管理 SSH 密码、IP 地址和命令行命令等敏感操作数据时遇到的痛点，解决了传统剪贴板管理器臃肿和密码管理器速度慢的问题。Sklad 的创新之处在于将本地 JSON 数据结构无缝映射到操作系统的原生系统托盘菜单，实现无需打开窗口即可深度导航层级化数据的体验。它还采用了 AES-256-GCM 加密和 Argon2 密钥派生，确保数据安全，并在锁定后从内存中清除敏感信息。

**如何使用：**
作为一名开发者，你可以将 Sklad 安装在你的电脑上。首次使用时，你会创建一个安全的密码来加密你的数据仓库。接着，你可以开始在 Sklad 中创建不同的文件夹和子文件夹，并将你的 SSH 密码、API 密钥、常用的命令行指令（比如 `git clone --depth 1 https://github.com/user/repo.git`）或者特定的 IP 地址等信息以文本片段的形式保存进去。Sklad 会自动将这些分类好的信息呈现在系统托盘的菜单中。当你需要使用某个信息时，只需将鼠标悬停在 Sklad 的系统托盘图标上，层层展开菜单，找到你想要的内容，点击即可复制到剪贴板，然后粘贴到你的终端或其他应用程序中。这种方式比打开密码管理器或者搜索文本文件要快得多，尤其适合频繁操作的开发场景。
