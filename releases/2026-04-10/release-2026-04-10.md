### 1.FluidCAD: 代码驱动的 3D 设计引擎

![FluidCAD: 代码驱动的 3D 设计引擎](https://showhntoday.com/images/47721997.png)

**URL：** [https://fluidcad.io/](https://fluidcad.io/)

**作者：** maouida

**描述：**
FluidCAD 是一个革命性的参数化 CAD (计算机辅助设计) 工具，它允许用户通过编写 JavaScript 代码来创建 3D 模型。它摆脱了传统 CAD 软件繁琐的鼠标拖拽操作，专注于通过代码的逻辑和精度来驱动设计。其核心创新在于将强大的 OpenCASCADE.js (基于 WebAssembly) CAD 内核嵌入到 Web 环境中，并提供实时渲染、智能交互辅助和对设计“特征”而非仅仅形状的操控，极大地降低了复杂模型的设计门槛，同时保证了设计的精确性和可复用性。

**如何使用：**
开发者可以将 FluidCAD 集成到自己的 Web 应用中，或者直接使用其提供的 Web 界面进行设计。

**核心使用流程：**
1.  **编写代码：** 使用 JavaScript 编写代码来定义 3D 模型的形状、尺寸、位置和操作。例如，你可以编写代码来绘制一个圆形，然后将其拉伸成一个圆柱体。
2.  **实时预览：** FluidCAD 会在你编写代码的同时，实时渲染出 3D 模型，方便你随时查看设计效果。
3.  **交互辅助：** 当需要进行一些复杂的几何操作时，可以通过鼠标进行交互，FluidCAD 会将这些交互转化为代码的一部分。
4.  **导出模型：** 完成设计后，可以将模型导出为常见的 CAD 文件格式（如 STEP），方便在其他设计软件中使用。

**技术集成场景：**
*   **Web 端的 3D 定制化产品展示：** 比如在线定制家具、个性化配件等，用户可以通过简单的参数或选择，代码自动生成定制的 3D 模型。
*   **自动化建模工具：** 开发用于特定行业的自动化建模工具，例如根据参数自动生成建筑构件、机械零件等。
*   **教育与原型开发：** 帮助学生和初创团队快速学习 3D 建模原理，并用代码快速验证设计想法。

所以，它能让你用熟悉的编程方式，在 Web 上构建出精确、可控的 3D 模型，并轻松将其集成到你的产品或工作流中。

![Go密语保险箱（Keeper）](https://showhntoday.com/images/47715339.png)

**作者：** babawere
