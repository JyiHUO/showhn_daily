### 1.Craft: C/C++ 提速构建引擎

![Craft: C/C++ 提速构建引擎](https://showhntoday.com/images/47705413.png)

**URL：** [https://github.com/randerson112/craft](https://github.com/randerson112/craft)

**作者：** randerson_112

**描述：**
Craft 是一个为 C 和 C++ 开发者设计的轻量级构建和工作流工具，旨在解决项目初始化和依赖管理耗时的问题。它通过一个简单的 `craft.toml` 配置文件替代了复杂的 `CMakeLists.txt`，并能自动生成构建脚本，极大地简化了 C/C++ 项目的搭建过程，让开发者能更快地投入到核心代码开发中。

**如何使用：**
开发者在使用 Craft 时，可以首先通过 `craft init` 命令来初始化一个新的 C/C++ 项目，或者让 Craft 来管理一个已有的项目。项目根目录下的 `craft.toml` 文件是你与 Craft 沟通的唯一界面。在这里，你可以定义项目的基本信息，比如：

toml
[project]
name = "my_awesome_app"
version = "1.0.0"
language = "cpp"
cpp_standard = 17

[build]
type = "executable"


当你需要添加一个外部依赖时，只需运行 `craft add <dependency_source>`。例如，要添加一个 GitHub 上的库，可以执行 `craft add --git https://github.com/some/library --links my_lib`。Craft 会自动处理下载、配置和集成，让你能够专注于编写业务逻辑。

对于更复杂的配置，Craft 允许使用 `CMakeLists.extra.cmake` 文件进行补充，确保了灵活性。Craft 支持 macOS, Linux, 和 Windows 操作系统。
